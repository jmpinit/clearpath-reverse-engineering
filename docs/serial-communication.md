# Serial Communication

Up to 16 motors can communicate on the same bus.

## Speeds

* 9600
* 115200
* 230400
* 460800
* 921600
* 1036800

## Operation Timing

| operation     | time (ms) |
| ------------- | --------- |
| break         | 40        |
| write timeout | 1000      |
| read timeout  | 1000      |
| reset time    | 1200      |
